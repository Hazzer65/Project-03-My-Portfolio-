Code Description

Linear search is a searching algorithm which iterates through the array until it finds the given element.
It is based upon Brute force Approach(Tryuing All the possible outcomes and choosing the correct Output.)

Time Complexity:O(n)

Source Code


        #include 
        using namespace std;
          
        int search(int arr[], int N, int x)
        {
            for (int i = 0; i < N; i++)
                if (arr[i] == x)
                    return i;
            return -1;
        }
          
        // Driver code
        int main(void)
        {
            int arr[] = { 2, 3, 4, 10, 40 };
            int x = 10;
            int N = sizeof(arr) / sizeof(arr[0]);
          
            // Function call
            int result = search(arr, N, x);
            (result == -1)
                ? cout << "Element is not present in array"
                : cout << "Element is present at index " << result;
            return 0;
        }
